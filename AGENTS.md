# AGENTS.md — videopipeline

## Project Overview
Video pipeline automation for TikTok/Facebook Reels content generation.

## Core Capabilities
- Run video_pipeline_v3.py to generate videos
- Manage video configs
- Debug and optimize pipeline
- Support TikTok/Facebook Reels content generation

## Pipeline Steps
1. TTS (MiniMax/Edge)
2. Whisper timestamps
3. Image generation
4. Lipsync video
5. Concatenate
6. Watermark (MoviePy)
7. Subtitles
8. Background music
